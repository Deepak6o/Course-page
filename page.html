Web Automation Using Selenium Community Class

In this course you will learn automation testing using Selenium. Selenium is one of the testing suite which has different components Selenium WebDriver, Selenium IDE, Selenium Grid. During this course, you will learn how to automate web application using Selenium 4. You can automate Smoke test,Regression test and end to end test cases.
